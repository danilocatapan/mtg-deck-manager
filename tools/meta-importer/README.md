# Meta Importer (offline)

Script offline para coletar dados de meta EDH e gerar arquivos em `backend/src/main/resources/meta/commanders`.

## Uso

```bash
node import-edhrec.mjs --commander "Xenagos, God of Revels" --slug xenagos-god-of-revels
```

> Observação: nunca usar scraping em runtime da API.
